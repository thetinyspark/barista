# Barista

A caffeinated HTML5 canvas rendering engine which you can use for creating:

- Video games. 
- Amazing data visualization tools.
- RIA (rich internet applications) like drawing apps.
- Funky filters.
- And so much more! 

## Getting Started

Open a terminal and type in the following instructions:
```bash 
npm i @thetinyspark/moocaccino-barista
``` 

## Technical stack

- The entire project is written in Typescript: https://www.typescriptlang.org/.
- The unit test framework is Jasmine: https://jasmine.github.io/..
- The project uses webpack as module bundler: https://webpack.js.org/.
- The documentation is generated by typedoc: https://typedoc.org/.
- The project uses Karma as module launcher: https://www.npmjs.com/package/karma

## Documentation

- The technical documentation is in the "docs" directory.
